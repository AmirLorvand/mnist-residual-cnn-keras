# MNIST Residual CNN with Keras

This repository contains my deep learning coursework project for the **Pattern Recognition, Neural Networks and Deep Learning** module at King’s College London.

The project focuses on handwritten digit classification using the MNIST dataset. I implemented and trained a convolutional neural network with residual blocks using TensorFlow/Keras.

* **Originally completed:** 2025
* **Published on GitHub:** 2026
* **Context:** MSc Artificial Intelligence Pattern Recognition, Neural Networks and Deep Learning coursework project

This repository has been cleaned and documented for portfolio purposes.

---

## Project Overview

The aim of this project was to build and train a deep neural network capable of classifying handwritten digits from the MNIST dataset.

The model was trained using TensorFlow/Keras and evaluated on the standard MNIST test set. The coursework also assessed how well the trained model generalised to an unseen handwritten digit test set.

---

## Model Architecture

The notebook implements a residual-style CNN for digit classification.

Main components include:

* input shape: `28 × 28 × 1`
* data augmentation
* convolutional layers
* batch normalisation
* ReLU activation
* residual blocks
* downsampling residual blocks
* global average pooling
* dropout
* softmax output layer

---

## Training Strategy

The training pipeline includes:

* MNIST loading and preprocessing
* train/validation/test split
* pixel normalisation to `[0, 1]`
* one-hot encoding of class labels
* data augmentation using rotation, translation, and zoom
* Adam optimiser
* cosine learning-rate decay
* categorical cross-entropy with label smoothing
* model checkpointing
* early stopping
* validation using hard augmentation
